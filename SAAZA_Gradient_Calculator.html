Gradient & Glideslope Calculator

Decimal/percent grade, line-level data reduction, longitudinal profile checks, and glideslope obstacle clearance — with the work shown.

Decimal & percent grade
Decimal grade = rise ÷ run  •  percent grade = decimal × 100. Always carry the +/− sign (+ = uphill, − = downhill). (e.g. +2.5' rise over 100' run = +2.5%)
Criteria (DAFMAN 13-217, Tbl 4.2): Longitudinal ≤ ±3% cumulative (max change ±1.5% per 200'). Transverse ±0.5% to 3% from centerline.
Line-level reduction — 200' section
Enter the rise (in inches, signed) read for each 20' line-level segment. Each 20' line = 240" run, so decimal grade = rise" ÷ 240. The cumulative is the section grade = total rise ÷ total run. (leave a row blank if not measured)
Longitudinal limit: section / cumulative grade ≤ ±3%.

Longitudinal profile — whole runway
Enter the grade (%) of each consecutive 200' section, in order (signed). Checks both limits at once: overall cumulative ≤ ±3%, and the change between any two adjacent sections ≤ ±1.5%. (a 3000' runway = 15 sections)
Both longitudinal limits: cumulative ≤ ±3%  •  section-to-section change ≤ ±1.5% per 200'.
Glideslope — obstacle clearance
Glideslope ratio = run ÷ rise, written X:1. Enter the distance to the obstacle and its height above the surface origin (end of overrun / usable). The approach–departure surface rises 1 ft for every X ft of run. (e.g. 6000' to a 2000' antenna = 3:1)
Required glideslope (Imaginary Surfaces): C-130 = 35:1  •  C-17 = 20:1. An obstacle clears if its ratio is ≥ the required ratio (i.e. it sits below the surface).
