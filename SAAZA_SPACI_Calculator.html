SPACI Calculator

Semi-prepared Airfield Condition Index — scores surface condition Green / Amber / Red

How it works: Inspect the airfield section, rate each of the 8 distress types as Green / Amber / Red, and this tool sums the deduct values, counts q, and helps you find the SPACI. Deduct values and severity criteria are from TSPWG 3-260-03.02-19 (Figures 2-37, 2-38, Table 2-10). The final SPACI is read from the authoritative curve/tables shown with your results, so the score always traces back to the source.
